Bee-Smart
==========================================================

Bee-Smart es una solución integral diseñada para modernizar y optimizar la gestión de actividades apícolas. Esta aplicación móvil permite a los apicultores llevar un control detallado y digitalizado de sus apiarios, facilitando la toma de decisiones basada en datos reales sobre la salud de las colmenas y la eficiencia de la producción.


Requisitos Previos
------------------

Para ejecutar este proyecto, asegúrate de tener instalados los siguientes componentes:

1. Git: Herramienta de control de versiones para clonar el repositorio.

2.  Node.js: Versión 20.15.0 LTS. (o superior)

3.  Expo Go: Instalado en tu dispositivo móvil (disponible en App Store o Google Play).

* * * * *

Guía de Instalación y Ejecución
-------------------------------

Sigue estos pasos para poner en marcha la aplicación en tu entorno local:

1.  Clonar el repositorio: 
```bash
git clone https://github.com/Zomofer/Bee-Smart.git
```
2.  Entrar al directorio del proyecto:
```bash
cd Bee-Smart
```
3.  Instalar las dependencias necesarias:
```bash
npm install
```
4.  Iniciar el servidor de desarrollo de Expo:
```bash
npx expo start
```
5.  Visualizar la aplicación: Una vez que el servidor esté corriendo, verás un código QR en tu terminal. Abre la aplicación Expo Go en tu dispositivo móvil, selecciona la opción para escanear el código QR y escanea el código mostrado en la pantalla de tu computadora.
